1、create a new repository on the command line	(创建一个新的库)
	touch README.md
	git init
	git add README.md
	git commit -m "first commit"
	git remote add origin https://github.com/zhongjixiuxing/documents.git
	git push -u origin master

2、push an existing repository from the command line（已存在的库更新到Github上）
	git remote add origin https上://github.com/zhongjixiuxing/documents.git
	git push -u origin master

3、如果本地的文件删除或者位置已经变动，想要更新到服务器，可以使用下面的指令
	git commit -a -m "your update message"
	git push origin master(分支名)


4、如果对文件作出修改，需要更新服务端
	git add .(.代表当前所有的文件，如果需要可以指定更新的文件)
	git commit -m '更新信息'
	git push origin master(分支名)
